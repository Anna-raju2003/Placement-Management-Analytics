# Placement-Management-Analytics
# Campus Placement Analytics Dashboard — BuildX Project

## 📌 Project Overview
This project focuses on executing an end-to-end data analytics workflow on historical campus placement data. The primary business objective is to uncover key academic, demographic, and corporate experience drivers that influence student hiring outcomes and compensation packages.

## 🛠️ Tech Stack & Workflow
1. **Data Cleaning & Engineering (Python / Pandas):** Handled missing data values, mapped structural headers, and engineered academic performance tiers.
2. **Business Database Analysis (SQL):** Imported the processed dataset into a relational database to run aggregate, filter, and conditional conversion rate queries.
3. **Executive Visualization (Power BI):** Developed a dynamic, interactive multi-page dashboard analyzing recruitment trends and cross-metric correlations.

## 📁 Repository Directory Structure
* `cleaned_dataset.csv` : The structured, processed dataset utilized for analytics.
* `queries.sql` : Production-ready SQL scripts containing business intelligence queries.
* `dashboard.pbix` : The complete Power BI Desktop dashboard file.

## 📊 Core Business Insights Uncovered
* **The Experience Premium:** Candidates possessing prior corporate work experience achieved an 86.2% placement conversion rate, outperforming inexperienced peers by a 32.1% margin.
* **Stream Performance:** Commerce & Management graduates represent the largest student volume pipeline, locking in a standard entry-level salary benchmark of INR 278,600.
* **Academic Thresholds:** Over 91% of candidates sitting within the "High Distinction" performance bracket successfully converted interviews into active job offers.
